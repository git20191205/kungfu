### 自我介绍

"我有 4 个月的 Java 后端学习经验，完成了 9 个项目，其中最核心的是秒杀系统，QPS 达到 6711，P99 延迟 28ms。我熟悉 Spring Cloud 微服务架构，掌握 Redis、Kafka 等中间件，有完整的系统设计能力。"

### 项目介绍

"秒杀系统是一个高并发项目，我设计了四层过滤架构：CDN 过滤 50%、Gateway 限流 5000 QPS、Redis Lua 原子扣库存、Kafka 异步创建订单。通过三道防线防止超卖：Redis Lua + MySQL 乐观锁 + 唯一索引。最终实现 QPS 6711，P99 延迟 28ms，成功率 100%。"

### 技术深度

"在 JVM 调优方面，我使用 G1 GC，固定堆大小 8GB，MaxGCPauseMillis 50ms，实现了 GC 停顿小于 50ms。在并发优化方面，我使用 Redis Lua 保证原子性，布隆过滤器防止重复购买，线程池异步处理。"
