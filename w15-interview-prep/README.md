## 专题详情

### JVM 专题（20 题）
- **内存模型**（5 题）：堆栈、OOM、内存泄漏、引用类型、逃逸分析
- **垃圾回收**（5 题）：GC 算法、GC 器、G1 原理、Full GC、GC 调优
- **类加载**（5 题）：加载过程、双亲委派、打破委派、生命周期、热部署
- **性能调优**（5 题）：监控工具、CPU 100%、内存泄漏、参数优化、实战经验

### 并发专题（20 题）
- **线程基础**（5 题）：生命周期、创建方式、sleep/wait、停止线程、守护线程
- **锁机制**（5 题）：synchronized、ReentrantLock、死锁、volatile、CAS
- **并发工具**（5 题）：CountDownLatch、Semaphore、ThreadLocal、线程池、拒绝策略
- **并发容器**（5 题）：ConcurrentHashMap、CopyOnWriteArrayList、BlockingQueue、AQS、实战经验

### Spring 专题（20 题）
- **IOC 容器**（5 题）：IOC 概念、初始化过程、Bean 作用域、生命周期、@Autowired vs @Resource
- **AOP 切面**（5 题）：AOP 概念、实现原理、通知类型、切点表达式、失效场景
- **事务管理**（5 题）：传播行为、隔离级别、@Transactional 失效、手动控制、分布式事务
- **Spring Boot**（5 题）：自动配置、启动流程、自定义 Starter、配置优先级、实战经验

### 数据库专题（20 题）
- **MySQL 基础**（5 题）：存储引擎、索引类型、聚簇索引、B+Tree、覆盖索引
- **SQL 优化**（5 题）：慢查询、EXPLAIN、索引失效、分页优化、实战经验
- **事务与锁**（5 题）：ACID、隔离级别、MVCC、锁类型、死锁
- **Redis 缓存**（5 题）：数据类型、持久化、缓存问题、分布式锁、实战经验

### 分布式专题（20 题）
- **分布式理论**（5 题）：CAP、BASE、分布式事务、分布式 ID、分布式锁
- **微服务**（5 题）：优缺点、注册发现、负载均衡、限流、熔断
- **消息队列**（5 题）：作用、Kafka 架构、消息可靠性、消息顺序、实战经验
- **分布式缓存**（5 题）：Redis Cluster、一致性 Hash、缓存更新、缓存 Key、实战经验
