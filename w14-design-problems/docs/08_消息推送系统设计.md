# 消息推送系统设计

> 经典系统设计题：设计一个实时消息推送系统（WebSocket）

## 一、需求澄清（5W1H）

### 1.1 What - 做什么？

**核心功能**：
- 实时推送：服务器主动推送消息给客户端
- 在线状态：显示用户在线/离线状态
- 消息确认：确保消息送达
- 离线消息：用户离线时缓存消息

**示例场景**：
- 即时通讯（微信、钉钉）
- 系统通知（订单状态、物流更新）
- 实时协作（在线文档、白板）

### 1.2 Who - 谁用？

- 普通用户：接收消息
- 系统：推送通知

### 1.3 When - 什么时候？

- 7×24 小时实时推送

### 1.4 Where - 在哪里？

- 移动端（App）
- PC 端（Web）

### 1.5 Why - 为什么？

- 实时性：用户第一时间收到消息
- 用户体验：无需轮询，节省流量
- 业务需求：订单通知、聊天消息

### 1.6 How - 怎么做？

- 客户端建立 WebSocket 连接
- 服务器推送消息
- 客户端接收并展示

## 二、容量估算

### 2.1 假设条件

- 日活用户：1000 万
- 同时在线：100 万（10%）
- 每个用户每天收到 10 条消息
- 消息保存 30 天

### 2.2 连接数估算

```
同时在线：1,000,000
单机连接数：10,000（假设）
需要服务器：1,000,000 / 10,000 = 100 台
```

### 2.3 QPS 估算

```
每天消息数：10,000,000 × 10 = 100,000,000
平均 QPS：100,000,000 / 86400 = 1157 QPS
峰值 QPS：1157 × 3 = 3471 QPS
```

### 2.4 存储估算

```
单条消息：
- id: 8B
- from_user_id: 8B
- to_user_id: 8B
- content: 500B
- create_time: 8B
- 索引: 50B

单条记录：600B

每天消息数：100,000,000
保存 30 天：100,000,000 × 30 = 3,000,000,000（30 亿）

存储容量 = 600B × 3,000,000,000 × 2（副本）× 1.3（增长）
         = 4.7 TB
```

### 2.5 带宽估算

```
单条消息：600B
峰值 QPS：3471

带宽 = 3471 × 600B × 8 = 16.7 Mbps
```

### 3.2 技术选型

| 组件 | 技术 | 原因 |
|------|------|------|
| WebSocket | Netty | 高性能、支持百万连接 |
| 在线状态 | Redis | 高性能、支持 Pub/Sub |

| 注册中心 | Nacos | 服务发现 |

### 3.3 核心流程

**用户上线流程**：
```
1. 客户端建立 WebSocket 连接
2. 服务器分配连接 ID
3. 存储在线状态到 Redis（userId → serverId）
4. 拉取离线消息
5. 推送离线消息
```

**消息推送流程**：
```
1. 业务服务发送消息到 Kafka
2. 推送服务消费 Kafka 消息
3. 查询 Redis 获取用户所在服务器
4. 通过 RPC 调用目标服务器
5. 目标服务器推送消息给客户端
6. 客户端返回 ACK
7. 更新消息状态为已送达
```

### 4.2 WebSocket 服务器设计

**Netty 实现**：

```java
@Component
public class WebSocketServer {
    
    // 存储所有连接（userId → Channel）
    private static final Map<Long, Channel> CHANNELS = new ConcurrentHashMap<>();
    
    @PostConstruct
    public void start() {
        EventLoopGroup bossGroup = new NioEventLoopGroup(1);
        EventLoopGroup workerGroup = new NioEventLoopGroup();
        
        try {
            ServerBootstrap bootstrap = new ServerBootstrap();
            bootstrap.group(bossGroup, workerGroup)
                .channel(NioServerSocketChannel.class)
                .childHandler(new ChannelInitializer<SocketChannel>() {
                    @Override
                    protected void initChannel(SocketChannel ch) {
                        ch.pipeline()
                            .addLast(new HttpServerCodec())
                            .addLast(new HttpObjectAggregator(65536))
                            .addLast(new WebSocketServerProtocolHandler("/ws"))
                            .addLast(new WebSocketHandler());
                    }
                });
            
            ChannelFuture future = bootstrap.bind(8080).sync();
            log.info("WebSocket 服务器启动成功，端口：8080");
            future.channel().closeFuture().sync();
        } finally {
            bossGroup.shutdownGracefully();
            workerGroup.shutdownGracefully();
        }
    }
    
    // 添加连接
    public static void addChannel(Long userId, Channel channel) {
        CHANNELS.put(userId, channel);
        
        // 存储在线状态到 Redis
        redisTemplate.opsForHash().put("online:users", userId.toString(), getServerId());
        redisTemplate.expire("online:users", 30, TimeUnit.SECONDS);
    }
    
    // 移除连接
    public static void removeChannel(Long userId) {
        CHANNELS.remove(userId);
        redisTemplate.opsForHash().delete("online:users", userId.toString());
    }
    
    // 推送消息
    public static void push(Long userId, String message) {
        Channel channel = CHANNELS.get(userId);
        if (channel != null && channel.isActive()) {
            channel.writeAndFlush(new TextWebSocketFrame(message));
        }
    }
}
```

**WebSocket 处理器**：

```java
public class WebSocketHandler extends SimpleChannelInboundHandler<TextWebSocketFrame> {
    
    @Override
    protected void channelRead0(ChannelHandlerContext ctx, TextWebSocketFrame msg) {
        String text = msg.text();
        Message message = JSON.parseObject(text, Message.class);
        
        switch (message.getType()) {
            case "CONNECT":
                // 用户上线
                handleConnect(ctx, message);
                break;
            case "HEARTBEAT":
                // 心跳
                handleHeartbeat(ctx, message);
                break;
            case "MESSAGE":
                // 发送消息
                handleMessage(ctx, message);
                break;
            case "ACK":
                // 消息确认
                handleAck(ctx, message);
                break;
        }
    }
    
    private void handleConnect(ChannelHandlerContext ctx, Message message) {
        Long userId = message.getUserId();
        
        // 1. 添加连接
        WebSocketServer.addChannel(userId, ctx.channel());
        
        // 2. 拉取离线消息
        List<Message> offlineMessages = messageService.getOfflineMessages(userId);
        
        // 3. 推送离线消息
        for (Message msg : offlineMessages) {
            ctx.channel().writeAndFlush(new TextWebSocketFrame(JSON.toJSONString(msg)));
        }
    }
    
    private void handleHeartbeat(ChannelHandlerContext ctx, Message message) {
        // 续期在线状态
        Long userId = message.getUserId();
        redisTemplate.expire("online:users", 30, TimeUnit.SECONDS);
        
        // 返回心跳响应
        ctx.channel().writeAndFlush(new TextWebSocketFrame("{\"type\":\"PONG\"}"));
    }
    
    @Override
    public void channelInactive(ChannelHandlerContext ctx) {
        // 用户下线
        Long userId = getUserId(ctx.channel());
        WebSocketServer.removeChannel(userId);
    }
}
```

### 4.3 消息推送设计

**推送服务**：

```java
@Service
public class PushService {
    
    @Autowired
    private RedisTemplate<String, Object> redisTemplate;
    
    @Autowired
    private RpcClient rpcClient;
    
    // 推送消息
    public void push(Long userId, Message message) {
        // 1. 查询用户所在服务器
        String serverId = (String) redisTemplate.opsForHash().get("online:users", userId.toString());
        
        if (serverId != null) {
            // 2. 用户在线，RPC 调用目标服务器
            rpcClient.call(serverId, "push", userId, message);
        } else {
            // 3. 用户离线，存储离线消息
            messageService.saveOfflineMessage(userId, message);
        }
    }
    
    // 批量推送
    public void batchPush(List<Long> userIds, Message message) {
        // 按服务器分组
        Map<String, List<Long>> serverUsers = new HashMap<>();
        
        for (Long userId : userIds) {
            String serverId = (String) redisTemplate.opsForHash().get("online:users", userId.toString());
            if (serverId != null) {
                serverUsers.computeIfAbsent(serverId, k -> new ArrayList<>()).add(userId);
            }
        }
        
        // 批量推送
        for (Map.Entry<String, List<Long>> entry : serverUsers.entrySet()) {
            rpcClient.call(entry.getKey(), "batchPush", entry.getValue(), message);
        }
    }
}
```

### 4.4 高可用设计

**心跳机制**：

```java
// 客户端每 10 秒发送心跳
setInterval(() => {
    ws.send(JSON.stringify({type: 'HEARTBEAT', userId: 10001}));
}, 10000);

// 服务器 30 秒未收到心跳，断开连接
@Scheduled(fixedRate = 30000)
public void checkHeartbeat() {
    long now = System.currentTimeMillis();
    for (Map.Entry<Long, Channel> entry : CHANNELS.entrySet()) {
        Channel channel = entry.getValue();
        Long lastHeartbeat = (Long) channel.attr(LAST_HEARTBEAT).get();
        
        if (now - lastHeartbeat > 30000) {
            // 超时，断开连接
            channel.close();
            removeChannel(entry.getKey());
        }
    }
}
```

**重连机制**：

```javascript
// 客户端自动重连
let reconnectCount = 0;
const maxReconnect = 5;

function connect() {
    ws = new WebSocket('ws://localhost:8080/ws');
    
    ws.onopen = () => {
        console.log('连接成功');
        reconnectCount = 0;
        // 发送连接消息
        ws.send(JSON.stringify({type: 'CONNECT', userId: 10001}));
    };
    
    ws.onclose = () => {
        console.log('连接断开');
        // 自动重连
        if (reconnectCount < maxReconnect) {
            reconnectCount++;
            setTimeout(connect, 1000 * reconnectCount);  // 指数退避
        }
    };
}
```

**消息确认机制**：

```java
// 1. 服务器推送消息，等待 ACK
public void push(Long userId, Message message) {
    Channel channel = CHANNELS.get(userId);
    if (channel != null) {
        // 推送消息
        channel.writeAndFlush(new TextWebSocketFrame(JSON.toJSONString(message)));
        
        // 等待 ACK（5 秒超时）
        CompletableFuture<Boolean> ackFuture = new CompletableFuture<>();
        ACK_MAP.put(message.getId(), ackFuture);
        
        try {
            Boolean ack = ackFuture.get(5, TimeUnit.SECONDS);
            if (ack) {
                // 更新消息状态为已送达
                messageService.updateStatus(message.getId(), MessageStatus.DELIVERED);
            }
        } catch (TimeoutException e) {
            // 超时，重试或存储离线消息
            log.warn("消息推送超时：{}", message.getId());
        }
    }
}

// 2. 客户端返回 ACK
ws.onmessage = (event) => {
    const message = JSON.parse(event.data);
    // 返回 ACK
    ws.send(JSON.stringify({type: 'ACK', messageId: message.id}));
};

// 3. 服务器收到 ACK
private void handleAck(ChannelHandlerContext ctx, Message message) {
    Long messageId = message.getMessageId();
    CompletableFuture<Boolean> ackFuture = ACK_MAP.remove(messageId);
    if (ackFuture != null) {
        ackFuture.complete(true);
    }
}
```
