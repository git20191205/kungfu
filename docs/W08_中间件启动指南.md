# W08 中间件启动指南（最小化配置）

## 前提

已确认可用：MySQL(3306) + Redis(6379) + Kafka(9092)

只需额外启动：Nacos + Seata

---

## 1. Nacos 2.2.3

### 下载

https://github.com/alibaba/nacos/releases/tag/2.2.3

下载 `nacos-server-2.2.3.zip`，解压到任意目录（如 `D:\nacos`）

### 启动（单机模式）

```bash
cd D:\nacos\bin
startup.cmd -m standalone
```

### 验证

浏览器打开 http://localhost:8848/nacos

账号密码：nacos / nacos

看到控制台页面即成功。

### 常见问题

启动失败看日志：`D:\nacos\logs\start.out`

如果端口冲突：编辑 `conf/application.properties`，改 `server.port=8848`

---

## 2. Seata 1.7.1

### 下载

https://github.com/apache/incubator-seata/releases/tag/v1.7.1

下载 `seata-server-1.7.1.zip`，解压到任意目录（如 `D:\seata`）

### 最小化配置

编辑 `D:\seata\conf\application.yml`，替换为以下内容：

```yaml
server:
  port: 7091

spring:
  application:
    name: seata-server

logging:
  config: classpath:logback-spring.xml
  file:
    path: ./logs
console:
  user:
    username: seata
    password: seata

seata:
  config:
    type: file
  registry:
    type: file
  store:
    mode: db
    db:
      datasource: druid
      db-type: mysql
      driver-class-name: com.mysql.cj.jdbc.Driver
      url: jdbc:mysql://localhost:3306/seata_server?useSSL=false&allowPublicKeyRetrieval=true&serverTimezone=Asia/Shanghai
      user: root
      password: "123456"
      min-conn: 5
      max-conn: 30
  security:
    secretKey: SeataSecretKey0123456789012345678901234567890123456789012345678901234
    tokenValidityInMilliseconds: 1800000
```

### 建库建表

在 MySQL 中执行：

```sql
CREATE DATABASE IF NOT EXISTS seata_server DEFAULT CHARACTER SET utf8mb4;
USE seata_server;

-- Seata Server 需要的 3 张表
CREATE TABLE IF NOT EXISTS global_table (
    xid                       VARCHAR(128)  NOT NULL,
    transaction_id            BIGINT,
    status                    TINYINT       NOT NULL,
    application_id            VARCHAR(32),
    transaction_service_group VARCHAR(32),
    transaction_name          VARCHAR(128),
    timeout                   INT,
    begin_time                BIGINT,
    application_data          VARCHAR(2000),
    gmt_create                DATETIME,
    gmt_modified              DATETIME,
    PRIMARY KEY (xid),
    KEY idx_status_gmt_modified (status, gmt_modified),
    KEY idx_transaction_id (transaction_id)
) ENGINE = InnoDB DEFAULT CHARSET = utf8mb4;

CREATE TABLE IF NOT EXISTS branch_table (
    branch_id         BIGINT        NOT NULL,
    xid               VARCHAR(128)  NOT NULL,
    transaction_id    BIGINT,
    resource_group_id VARCHAR(32),
    resource_id       VARCHAR(256),
    branch_type       VARCHAR(8),
    status            TINYINT,
    client_id         VARCHAR(64),
    application_data  VARCHAR(2000),
    gmt_create        DATETIME(6),
    gmt_modified      DATETIME(6),
    PRIMARY KEY (branch_id),
    KEY idx_xid (xid)
) ENGINE = InnoDB DEFAULT CHARSET = utf8mb4;

CREATE TABLE IF NOT EXISTS lock_table (
    row_key        VARCHAR(128) NOT NULL,
    xid            VARCHAR(128),
    transaction_id BIGINT,
    branch_id      BIGINT       NOT NULL,
    resource_id    VARCHAR(256),
    table_name     VARCHAR(32),
    pk             VARCHAR(36),
    status         TINYINT      NOT NULL DEFAULT 0,
    gmt_create     DATETIME,
    gmt_modified   DATETIME,
    PRIMARY KEY (row_key),
    KEY idx_branch_id (branch_id),
    KEY idx_xid (xid)
) ENGINE = InnoDB DEFAULT CHARSET = utf8mb4;

CREATE TABLE IF NOT EXISTS distributed_lock (
    lock_key       CHAR(20)    NOT NULL,
    lock_value     VARCHAR(20) NOT NULL,
    expire         BIGINT,
    PRIMARY KEY (lock_key)
) ENGINE = InnoDB DEFAULT CHARSET = utf8mb4;

INSERT IGNORE INTO distributed_lock (lock_key, lock_value, expire) VALUES ('AsyncCommitting', ' ', 0);

```

### 启动

```bash
cd D:\seata\bin
seata-server.bat -p 8091
```

### 验证

浏览器打开 http://localhost:7091

账号密码：seata / seata

看到 Seata 控制台即成功。
